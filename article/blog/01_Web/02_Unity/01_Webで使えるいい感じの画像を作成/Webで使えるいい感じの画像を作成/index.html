Webで使えるいい感じの画像の作成

↑こんな画像を作りたい


経緯

Webサイトの画像を作成するにおいて、実際に撮影する環境がなく、CGで作成する方法もあまり知らなかった。

自分の知っている知識では、「Unity」を利用することで、簡単に作成できると考えた。


手順

  • Unityプロジェクト作成。
  • シーンにモデルを並べて、スクリーンショット

Unityインストール

Unity Hubをダウンロードして、GoogleアカウントでPersonalのライセンスを取得


シーンの作成

Unityプロジェクトを作成。ここで、High Definition Render Pipeline（高解像度レンダリングパイプライン）を選択する。

HDRPでは、カメラに、ポストエフェクト(現実のカメラの効果)を簡単に設定できる。


シーンにモデルを並べる

HDRPでは、標準でサンプルのシーンがあり、床や壁があったため、それらのマテリアルを利用しました。

スマートフォンと、タブレットを意識して作成しました。モデリングは行わず、Unityで作成できる"Cube"を利用したため、角が角ばっています。


スマートフォンやタブレットに映し出された画像は、私が作成したポートフォリオ01です。
