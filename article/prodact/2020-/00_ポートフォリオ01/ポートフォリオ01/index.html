ポートフォリオ 01


ポートフォリオ01 リンク


はじめてWebサイトを作成した

就職活動で、初めて会社に応募した際、「可能であればポートフォリオの提出」となっていたため、Webサイトの制作の練習もかねて、制作した。


特徴

  • レスポンシブ対応


制作において

このとき、デザインに関しては、制作しながら、よく見るサイト構成を参考にして制作した。

制作に利用したツール

  • Visual Studio Code
  • Git Hub

さいごに

ハローワークから「紹介状」を貰ってから、ポートフォリオの制作を始めたため、時間がなかった（言い訳です）。

限られた時間で、多くの環境で正しく閲覧できるサイトの制作を目指したため、デザインについておろそかになってしまった（まあ、これでいいかと思ってしまった）。

このとき、初めて応募した会社では、面接で採用を見送られた。
自分でも自覚していたが、面接担当の方から、デザインについての知識が不足していると指摘を頂いたため、ポートフォリオ02を制作することにした。
